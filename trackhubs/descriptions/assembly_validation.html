Description

Tracks used for assembly validation.

Methods

PacBio HiFi reads were aligned to the assembly using minimap2 with the parameters -x lr:hq --secondary=no. Oxford Nanopore (ONT) reads were aligned using winnowmap with the parameters -x map-ont --secondary=no. Alignments were subsequently filtered with samtools to retain reads with mapping quality MQ >= 1 and to remove alignments with flag 3840.

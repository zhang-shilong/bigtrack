Description

Repeat elements identified by RepeatMasker and RepeatModeler.

Display Conventions and Configuration

The following table lists the repeat class colors:

Color Repeat Class
SINE - Short Interspersed Nuclear Element
LINE - Long Interspersed Nuclear Element
LTR - Long Terminal Repeat
DNA - DNA Transposon
Simple - Single Nucleotide Stretches and Tandem Repeats
Low_complexity - Low Complexity DNA
Satellite - Satellite Repeats
RNA - RNA Repeats (including RNA, tRNA, rRNA, snRNA, scRNA, srpRNA)
Other - Other Repeats (including class RC - Rolling Circle)
Unknown - Unknown Classification

A "?" at the end of the "Family" or "Class" (for example, DNA?) signifies that the curator was unsure of the classification. At some point in the future, either the "?" will be removed or the classification will be changed.

Methods

RepeatMasker was run on the entire genome using the NCBI/RMBLAST search engine, with the Dfam and RepBase databases in sensitive mode (-s). The genome was then hard-masked using BEDTools and used as input for RepeatModeler with the long-terminal repeat structural discovery pipeline. RepeatMasker was subsequently run on the hard-masked genome using the repeat database generated by RepeatModeler. The repeats from both runs were then combined.
